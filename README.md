# 1️⃣ Menu-Driven Marks Analyser

## Description
A menu-based Python application that allows users to enter student marks and perform various analyses.

## Features
- Add student marks
- Calculate total marks
- Calculate average marks
- Find highest marks
- Find lowest marks
- Display all marks

## Concepts Used
- Lists
- Loops
- Functions
- Conditional Statements
- Menu-Driven Programming

## Sample Output

```
1. Add Marks
2. View Marks
3. Average Marks
4. Highest Marks
5. Lowest Marks
6. Exit
```


### Run Any Project

```bash

```

or

